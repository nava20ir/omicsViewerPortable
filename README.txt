To explore the data, double click on run(.bat), the viewer will be opened in your default browser.
Dataset can be selected from the text box on the top-right corner. demo.RDS is for demostration purpse. 

To create data package and share with others:
  1. save the prepared object (the file name needs to be end with .RDS and the config.yaml file(optional)) and put it in the "appData" folder.
  You can also delete the demo data (demo.RDS) if you want.
  2. zip the "omicsViewer_x.x.x" folder and share with other. 
  3. To start omicsViewer, double click on run(.bat). The viewer will be opened in your default browser.
